## Logan's Taco
I present my first website created in TypeScript. This restaurant's website is inspired by the character of Logan Kapuchino played by the streamer Paris Platynov on Twitch.

# Technologies
- JavaScript
- React.js
- Next.js
- MongoDB
- CSS

# Libraries
- Nodemailer
- VanillaTilt

# Link
